# Ejecutar Moodle mediante docker compose

# Ejecutar localmente usando docker 

1. Ejecutar los siguientes comandos:

   $ git clone https://github.com/Tuyub22/MoodleRJ.git
   $ cd MoodleRJ/

2. Levantar los contenedores

  $ docker-compose up
